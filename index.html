¿Qué es Validador de RUT?

Validador de RUT es un pequeño script escrito con HTML, JavaScript y CSS para validar los RUT de Chile en base al método de validación de RUT del "Módulo 11".

¿Qué es el Módulo 11?

El "módulo 11" es un método de valicación de RUT en Chile que se ocupa para verificar si el "resto" del cuerpo del RUT coincide con el dígito verificador (El Número que va luego del guión).

¿Cómo funciona?

Básicamente el usuario debe ingresar un RUT en el Input de Texto que se encuentra en la tarjeta de arriba y el Script JS analizará los datos mediante se van ingresando los dígitos.

Si el RUT es menor a 7 dígitos, se considera que el RUT es muy corto.

Si el RUT no coincide con el dígito verificador, se considera que el RUT es Inválido.

Por el contrario, si el RUT coincide con el dígito verificador (Dígito luego del Guión) se considera que el RUT es válido.

El método Utilizado es el siguiente "Módulo 11".
